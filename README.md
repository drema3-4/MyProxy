# SingBox Manager

## О проекте
**SingBox Manager** — это графический инструмент для удобного управления клиентом [Sing-Box](https://github.com/SagerNet/sing-box) на Windows. Программа позволяет скачивать актуальные версии ядра, файлы геоданных (geosite, geoip), настраивать подключение к вашему VPS и запускать/останавливать туннель одним кликом.

> 💡 Всё, что делает эта программа, вы можете выполнить вручную через консоль (это описано в документации). Но мы создали простой интерфейс, чтобы сэкономить ваше время и нервы.
---
## Установка на VPS (серверная часть)
На вашем VPS должен быть установлен и настроен Sing-Box (как сервер). Если вы ещё этого не сделали, воспользуйтесь официальной документацией или нашими скриптами.

1. **Подключитесь к VPS** по SSH.
2. **Склонируйте репозиторий** (если вы хотите использовать примеры конфигураций):
   ```bash
   git clone https://github.com/yourusername/singbox-manager.git
   cd singbox-manager
   ```
3. **(Опционально) Запустите скрипт быстрой настройки сервера**:
   ```bash
   chmod +x setup_server.sh
   ./setup_server.sh
   ```
   Скрипт установит sing-box, создаст базовый конфиг и запустит сервис.

После настройки запишите параметры подключения (IP, порт, UUID, публичный ключ и т.д.) — они понадобятся при настройке клиента.

## Установка на компьютер (клиентская часть)
### 1. Клонирование репозитория (опционально)
Если вы хотите иметь исходники или примеры конфигов, выполните:
```bash
git clone https://github.com/yourusername/singbox-manager.git
```
В репозитории уже лежит архив `SingBoxManager.zip` с готовым исполняемым файлом.
### 2. Распаковка приложения
- Создайте папку для программы, например: `D:\sing-box\`
- Распакуйте содержимое `SingBoxManager.zip` в эту папку.
- В папке должны появиться:
  - `SingBoxManager.exe` – основная программа
  - (возможно) папка `data` для хранения конфигураций
### 3. Создание ярлыка (по желанию)
- Нажмите правой кнопкой на `SingBoxManager.exe` → **Отправить** → **Рабочий стол (создать ярлык)**.
- Теперь вы можете запускать программу прямо с рабочего стола.

## Первый запуск и настройка клиента
1. **Запустите `SingBoxManager.exe`**. Откроется главное окно с несколькими вкладками.
2. **Перейдите на вкладку «Скачивание»**:
   - Нажмите **«Скачать sing-box»** – программа загрузит последнюю версию ядра Sing-Box (под Windows) и сохранит его в папку `bin`.
   - Затем последовательно скачайте файлы геоданных:
     - **«Скачать geosite»**
     - **«Скачать geoip»**
   > Эти файлы необходимы для корректной маршрутизации трафика.
3. **Перейдите на вкладку «Настройки»**:
   - Введите параметры вашего VPS, полученные ранее:
     - Адрес сервера (IP или домен)
     - Порт
     - Тип протокола (например, VLESS, Trojan и т.д.)
     - UUID / Пароль
     - Дополнительные параметры (если требуются)
   - Нажмите **«Сохранить»** – программа создаст конфигурационный файл `config.json` в папке `data`.
4. **Перейдите на вкладку «Управление»**:
   - Нажмите **«Запустить»** – статус изменится на «Подключено».
   - Если всё настроено правильно, трафик вашего компьютера пойдёт через VPS.
   - Чтобы остановить туннель, нажмите **«Остановить»**.

## Структура проекта
```
singbox-manager/
│
├── SingBoxManager.exe          # Скомпилированное приложение
├── main.py                      # Исходный код (Python)
├── requirements.txt             # Зависимости Python
├── README.md                    # Этот файл
├── setup_server.sh              # Скрипт для быстрой настройки VPS (Linux)
│
├── data/                        # Папка для пользовательских данных
│   ├── config.json              # Конфигурация клиента (создаётся при сохранении)
│   └── ...                      # Логи, кеш
│
├── bin/                          # Исполняемые файлы ядра
│   ├── sing-box.exe             # Скачивается автоматически
│   └── ...                      # Вспомогательные файлы
│
└── geo/                          # Файлы геоданных (скачиваются)
    ├── geosite.db
    └── geoip.db
```

## Используемые технологии
- **Python 3.10+** – язык разработки
- **Tkinter** – графический интерфейс (встроен в Python)
- **PyInstaller** – создание автономного .exe файла
- **Sing-Box** – основное ядро для прокси
- **Requests** – загрузка файлов по HTTP
- **JSON** – хранение конфигураций

---
## Примечания
- При первом запуске антивирус может ругаться на `SingBoxManager.exe` – это нормально для самодельных сборок. Добавьте программу в исключения, если доверяете разработчику.
- Программа не требует прав администратора, но для работы с сетевыми интерфейсами Sing-Box может запросить их при запуске (в зависимости от конфигурации).
- Все загружаемые файлы проверяются по контрольным суммам (если доступны).
